Описание:
Программа реализует метод Ньютона и находит одно решение функции на заданном
промежутке с заданной точностью.

Установка:
python setup.py install

Запуск:
python newtonpackage\__main__.py [y, a, b, e] [-h, -ex, -t]

Запуск с примерами:
python newtonpackage\__main__.py [y, a, b, e] [-ex, --example]

Запуск тестов:
python newtonpackage\__main__.py [y, a, b, e] [-t, --testing]
